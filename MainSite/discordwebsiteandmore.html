    Description

    This Plugin will allow you to display important server Url's such as dynmap, voting, server website, discord invites and more!

    #--------------------------------------------------------------------------
    # Plugin Permissions
    #--------------------------------------------------------------------------
    # dwm.default: (This permission must be given out to everyone, It will be used for everything you may want reg players to have)
    #
    # dwm.ping: (This will allow you to specifically choose who you want to be able to ping!)
    #
    # dwm.admin: (This perm will receive pings from /dwmping admin)
    #
    # dwm.mod: (This perm will receive pings from /dwmping mod)
    #
    # dwm.helper: (This perm will receive pings from /dwmping helper)

    Commands
    -----------------------------
    /dynurl - Displays Dynmap Url
    /voting - Displays all Voting Url's
    /website - Displays Website Url
    /discord - Displays Discord Invite
    /dwmping admin
    /dwmping mod
    /dwmping helper
