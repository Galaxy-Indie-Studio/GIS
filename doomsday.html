    Description

    Have you ever wanted to start a zombie survivor but couldn't find any decent or unabandoned plugins to assist you in that! I have that's one of the whole purposes behind this plugin. Hopefully by the time I am finished this plugin will be your one stop shop for a zombie apocalypse style survivor! Eventually with this plugin you will have disease, mutated zombies, a built in weapon system, built in thirst system, and anything else we can think of to make the best Zombie apocalypse style plugin! With all of this being said I also hope to have it 100% configurable and 100% configurable ingame meaning that you all can change every little aspect of the plugin to fit your "Zombie Apocalypse" Needs! If you all have any suggestions, issues, etc. you can reach us the best at discord! You can also always report issues on github as well!! All of the issues go straight to my discord :D :D :D :D https://discord.gg/PSZS7WnuG2 :)

    # -------------------------------------------------------------
    # Plugin Permissions
    #---------------------------------------------------------------
    # dd.default: (This will allow players access to all the default commands)
    # dd.*: (This will allow the player to perform all commands)\
    # dd.admin: (This is only used for the ping system, Don't assign to not receive anything)
    # dd.mod: (This is only used for the ping system, Don't assign to not receive anything)
    # dd.helper: (This is only used for the ping system, Don't assign to not receive anything)
    # dd.doc: (This allows players to use /dday doc)

    Regualar Commands
    ========================================
    Player Commands
    -----------------------------------------
    /dday suicide
    /dday doc

    Ping Commands
    -----------------------------------------
    /dday ping admin
    /dday ping mod
    /dday ping helper
    /dday ping all

    Admin Commands
    ========================================

    Player Config Set Commands
    --------------------------------------

    /ddayadmin config set player killreward
    /ddayadmin config set player killmsg

    Zombie Config Set Commands
    ----------------------------------------

    /ddayadmin config set zombie name
    /ddayadmin config set zombie health
    /ddayadmin config set zombie strength
    /ddayadmin config set zombie killreward
    /ddayadmin config set zombie killmsg

    Player Config Enable Commands
    ----------------------------------------

    /ddayadmin config enable player killreward
    /ddayadmin config disable player killreward

    Zombie Config Enable Commands
    --------------------------------------------
    /ddayadmin config enable zombie killreward
    /ddayadmin config disable zombie killreward

    Show Commands
    ---------------------------------------------
    /ddayadmin config show all
    /ddayadmin config show settings
    /ddayadmin config show enabled
    /ddayadmin config show blocked

    Block Commands
    -----------------------------------------------
    /ddayadmin config block passive (MobName)
    /ddayadmin config block neutral (MobName)
    /ddayadmin config block hostile (MobName)
    /ddayadmin config block all
